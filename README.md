# Steps to Run Script
- Step 1
  * run command:
  ```
  git clone https://github.com/HashirAhmedd/Article-Script.git
  ```
- Step 2
  * inside Article-Script folder run command
  ```
  npm install
  ```
- Step 3
  * create file .env

- Step 4
  * inside .env file create enviroment variables like given in .env.sample

- Step 5
  * run main.js script
